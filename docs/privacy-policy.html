Политика конфиденциальности

Приложение «ЛетиУмно»

Дата обновления: 2 июля 2026 г.

1. Общие положения

Настоящая Политика конфиденциальности разработана в соответствии с Федеральным законом от 27.07.2006 №152-ФЗ «О персональных данных».

Оператор персональных данных: ИП Зубков С.В. (далее — Разработчик).

Приложение: «ЛетиУмно» (далее — Приложение).

2. Какие данные мы собираем

Приложение может собирать следующие данные:

  • Поисковые запросы — маршруты, даты, количество пассажиров. Используются исключительно для выполнения поиска авиабилетов и отелей. НЕ сохраняются на серверах после обработки.
  • Локальные данные — история поисков. Хранятся ТОЛЬКО на вашем устройстве.
  • Настройки приложения (язык, тема оформления) — хранятся локально на устройстве.

Приложение НЕ собирает: ФИО, адрес, телефон, email, геолокацию, данные банковских карт, рекламные идентификаторы.

3. Цели обработки данных

Обработка данных осуществляется исключительно для:

  • Поиска авиабилетов и отелей по запросу пользователя
  • Предоставления рекомендаций по планированию путешествий
  • Сравнения цен среди авиакомпаний и систем бронирования
  • Улучшения качества работы приложения

4. Хранение и защита данных

  • Все данные хранятся исключительно на устройстве пользователя в локальной базе данных.
  • Передача поисковых запросов осуществляется по зашифрованному каналу (HTTPS/TLS).
  • Запросы обрабатываются в оперативной памяти сервера и не записываются на диск.
  • Разработчик не имеет доступа к локальным данным пользователя.

5. Передача данных третьим лицам

Поисковые запросы передаются партнёрским сервисам (Travelpayouts) исключительно для поиска авиабилетов и отелей.

  • Данные не сохраняются на сторонних серверах после обработки.
  • Разработчик не передаёт, не продаёт и не предоставляет данные пользователей третьим лицам.
  • Обезличенная статистика может использоваться для улучшения сервиса.

6. Права пользователя (ст. 14 №152-ФЗ)

В соответствии с Федеральным законом №152-ФЗ «О персональных данных» вы имеете право:

  • Получить информацию об обработке ваших данных;
  • Потребовать удаления ваших данных;
  • Отозвать согласие на обработку данных.

Для удаления всех данных достаточно удалить приложение или очистить данные приложения в настройках устройства.

7. Файлы cookie и аналитика

Приложение не использует файлы cookie и рекламные трекеры. Приложение не требует регистрации.

8. Возрастные ограничения

Приложение предназначено для лиц старше 12+ лет.

9. Изменения политики

Разработчик оставляет за собой право обновлять настоящую Политику. Актуальная версия всегда доступна в приложении и по адресу: https://zubcoder.github.io/Lets-Go-Travel/privacy-policy.html
